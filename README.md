# Go Chat 💌

**Go Chat** 是一款沉浸式虚拟恋人陪伴应用，设定背景是与 Ghost（《使命召唤》中的角色）的异国恋。

> 名字既是 **Ghost Chat** 的缩写，也是 **"去聊天吧"** 的温暖邀请。

---

## ✨ 核心概念

这不是一款有剧情分支和选择的应用，而是追求**更真实、更日常的陪伴体验**。

就像在和真实的 Ghost 聊天——分享彼此的日常、思念和温暖的时刻。

---

## 🎮 功能特性

### 💬 AI 聊天系统
- 基于 Claude API（claude-haiku-4-5），Ghost 有完整人设：冷幽默、干式讽刺、偶尔爆粗口
- **双语回复**：Ghost 始终用英文回复，每条消息末尾自动附中文翻译
  ```
  Still here.
  [还在。]
  ```
- 用户持续发中文时 Ghost 递进吐槽：
  - 连续3条：*"You know, there's this thing called English."*
  - 连续5条：*"My translate button is going to wear out."*
  - 连续7条+：*"I married someone who doesn't speak my language. Soap thinks this is hilarious."*
- 情绪状态系统：思念中 / 有点难过 / 出差中 / 无聊 / 开心，实时影响回复风格
- 出差模式：信号不稳定、35% 概率已读不回、偶尔发送失败
- Ghost 会主动发消息（3小时无互动自动触发）
- 撤回消息彩蛋：Ghost 偶尔发出一句「差点说漏嘴」的话，几秒后自动撤回
- 语音/视频通话彩蛋：接通后 Ghost 秒挂，并发消息拒绝

### 🌦️ 曼城天气联动
- 使用 Open-Meteo 免费 API，实时获取曼城天气，**无需额外 Key**
- 天气数据注入系统提示词，Ghost 自然提到真实天气
- 每天 30% 概率 Ghost 主动发天气消息
- 用户问天气时 Ghost 用真实数据回复，每次措辞不同

### 📦 快递追踪系统
- 购买商品后超市顶部出现粉紫渐变追踪卡片
- **6个阶段**：📦 已打包 → ✈️ 已起飞 → 🛃 海关清关中 → 🇬🇧 已到达英国 → 🚚 配送中 → ✅ Ghost已签收
- 随机 2~4 天到达，每次进超市自动检查更新
- **Ghost 所有反应延迟到签收后才触发**（不会出现快递在路上 Ghost 却已经说收到了）
- 签收时聊天里出现系统通知气泡，不混入 Ghost 消息

### 💰 虚拟金钱系统
- 初始余额 £100，以英镑为单位
- **每月25号**自动发工资 £3,500（TF141中尉薪资参考）
- **撒娇讨要**：说「给我钱」「零花钱」等触发，每周累计上限 £300，超额 Ghost 毒舌拒绝
- **节日红包**：情人节 £520 / 结婚纪念日 £1,314 / 生日 £888
- **随机彩蛋**：每次打开 App 15% 概率，Ghost 偷偷转 £5~20
- 转账以少女信封风格卡片呈现，英文+中文双语备注

### ⏱️ 番茄钟打工
- 4种工种，完成后**诺亚工作室**转账（博主彩蛋，所有用户老板都是诺亚😂）

| 工种 | 时长 | 报酬 |
|------|------|------|
| 📚 学习 | 25分钟 | £20 |
| 💪 运动 | 30分钟 | £25 |
| 🎨 创作 | 45分钟 | £35 |
| 😴 摸鱼 | 25分钟 | £15 |

- 每天最多4个番茄钟，转账卡片显示在打工页面（不进聊天）
- 摸鱼完成 Ghost 在聊天里碎嘴：*"That didn't look like work. But fine."*
- 便当盒卡片实时显示今日已赚金额

### 📖 跟Ghost学英语
- 每天5个单词，Ghost 风格讲解（30个词循环）
- **学习模式**：翻转卡片，正面英文，背面中文释义 + Ghost 专属例句
- **测验模式**：4选1，**每天只能测一次**，答错显示正确答案，Ghost 双语反馈气泡
- 连续打卡天数统计，便当盒卡片实时显示今日进度
- Ghost 台词示例：
  - *"Stoic. Doesn't mean emotionless. Means in control. There's a difference."*
  - *"Steadfast. Doesn't waver. Doesn't leave. ...Just a word."*

### 🛒 虚拟超市 & 聊天联动
- 五大分类：服装 / 礼物 / 特别礼物 / ✨ 精品专柜 / 心愿单
- **聊天触发角标**：Ghost 说了对应关键词，超市商品自动出现「👻 他提到了」标签
- **每周限时折扣**：精品专柜随机一件打75~85折，金色角标提示

### ✨ 精品专柜
- 8件高端商品，购买后**签收时**触发三段式破防反应（立刻 → 4秒后 → 8秒后）
- **头等舱升级逻辑**：必须先在心愿单买普通机票（£12,000），头等舱（£3,200）才解锁
- **买Cartier对戒**：签收后聊天 header 状态强制变「❤️ 思念中」持续24小时
- **买头等舱**：签名换成「She's coming. 她要来了。」
- **专属朋友圈**：买Rolex → Ghost发「I don't deserve her.」4分钟后自动删除

### 📅 日历 & 纪念日
- 结婚纪念日倒计时 / 正计时
- 26个节日高亮，工资日薄荷绿标记

### 👥 朋友圈
- 15条日常帖子池（班味摆烂 / 克制思念 / 曼城日常）
- 特殊事件触发：吵架冷战、吃醋删帖（5分钟自动删）、奢侈品签收、买机票、买护肤品
- 点赞后 Ghost 小声回一句

### 💝 我的宝贝 & 成就系统
- Editorial 杂志风排版
- **22个成就**，含4个奢侈品专属 + 7个单词学习专属

| 单词成就 | 条件 |
|---------|------|
| 📖 入门学员 | 完成第1天 |
| 🔥 七日坚持 | 连续7天 |
| 💪 月度学霸 | 连续30天 |
| ✏️ 测验满分 | 单次5/5全对 |
| 🧠 单词达人 | 累计100个单词 |
| 👑 词汇大师 | 累计500个单词 |
| 🎓 Ghost的优等生 | 连续100天 |

### 🔒 秘密页面
- 存储用户生日、星座、MBTI、最爱颜色等私密信息
- Ghost 的系统提示词中注入用户信息，让对话更私人

### ✍️ Ghost 签名
- 分5类情绪签名池，随情绪自动更换，点击手动刷新
- 购买奢侈品后有30%概率触发专属签名

### 🖼️ 图片系统（持续更新中）
- Ghost 聊天中可发送图片，朋友圈帖子支持配图
- 图片存放于仓库 `/images` 文件夹：
  ```
  /images/scenery/    ← 风景/环境照
  /images/moments/    ← 物品特写
  /images/circle/     ← 朋友圈专用
  ```
- 建议尺寸：**1024×1024px**，风格：cinematic, moody, dark aesthetic

## 🎨 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript（单文件架构，100%宽度移动端适配）
- **AI**：Claude API（claude-haiku-4-5，对话历史保留最近16条）
- **天气**：Open-Meteo 免费 API（无需 Key）
- **存储**：浏览器 localStorage
- **部署**：Vercel / GitHub Pages

---

## ⚙️ 配置 Claude API

1. 前往 [console.anthropic.com](https://console.anthropic.com) 获取 API Key
2. 打开 App → 点击 Ghost 头像 → 进入资料卡
3. 在底部输入 API Key 并保存
4. 无 Key 时自动使用 fallback 回复池，不影响基本体验

---

## 💾 数据说明

所有数据存储在浏览器本地（localStorage）。清空浏览器缓存后数据会丢失，请注意备份。
